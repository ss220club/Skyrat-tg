dcs_check_list_arguments
/datum/unit_test/dcs_check_list_arguments

list arguments for bespoke elements are treated just like any other datum: as a text ref in the ID. Using un-cached lists in AddElement() and RemoveElement() calls will just create new elements over and over. That's what this unit test is for. It's not a catch-all, but it does a decent job at it.
